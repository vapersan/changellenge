Структура.
	приложение работающее на микросервисах можно пердставить в виде стены.
	проект - это фундамент, микросервисы - кирпичи.
	Кирпич абстрагированным с помощью бетона - среда.
	Каждому кирпичу - своё место в виде сервера и бд.
	Для чего?
	Вытаскиваем один кирпич из стены и... ничего не происходит. 
	Это называется независсимость. Для чего она тут нужна обсудим на следующем слайте.
		
Примеры использования.
	Один из самых популярных российских сайтов Вконтакте разработан по логикке микросервесной архитектуры.
	Это можно легко понять потому-что, если например выходит из строя сервер отвечающий за фотографии,
	то работает всё кроме этих самых фотографий. Проще говоря: сайт не знает откуда ему брать данные о фотографии
	и просто не отображает их. Та самая независсимость.

Hydra Search Micro
